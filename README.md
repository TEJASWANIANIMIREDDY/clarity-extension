🚀 Clarity – Instant Context & Action Assistant
🚀 Overview

Clarity is a Chrome extension that helps users instantly understand highlighted text and, more importantly, decide what to do next.

🎯 Problem

While browsing emails, articles, or study material, users often:

understand words but miss the actual intent
feel confused about what action to take next
waste time figuring out the next step
💡 Solution

Clarity allows users to highlight any text and get:

Meaning → simple explanation
Intent → what the text is trying to convey
Action → what the user should do next

All instantly, in a clean popup.

✨ Features
Highlight any text on any website
Smart breakdown: Meaning, Intent, Action
Clean popup UI
One-click trigger (Clarity button)
Works across emails, articles, and study content
⚙️ Installation
Download or clone the project
Open Chrome → chrome://extensions/
Enable Developer Mode
Click Load Unpacked
Select the project folder
